# SCANNER (PLCS-CW2)

**Scan a range of ports against your localhost.**

## Usage:
 ./scanner -s \<starting port\> -e \<ending port\>

 ## Options:
 -s \<starting port\> &ensp; range starting port <br />
 -e \<ending port\>   &ensp; range ending port <br />
 -h                   &ensp; display this help <br />
