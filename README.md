# SCANNER (PLCS-CW2)

**Scan a range of ports against your localhost.**

## Usage:
 ./scanner -s \<starting port\> -e \<ending port\>

 ## Options:
 -s \<starting port\>     range starting port <br />
 -e \<ending port\>       range ending port <br />
 -h                     display this help <br />
